# ⚡ Quick Deploy Steps - Maison Royale Residency

## 🚀 5-Minute Deployment Guide

### **1️⃣ Download Code**
- Figma Make → Download button → Save ZIP
- Extract files

### **2️⃣ GitHub Upload**
- github.com → New repository → `maison-royale-residency`
- "uploading an existing file" → Drag all files
- Commit changes

### **3️⃣ Vercel Import**
- vercel.com → Add New → Project
- Select `maison-royale-residency`
- Import

### **4️⃣ Environment Variables**
Add these 2 variables:

```
VITE_SUPABASE_URL
https://rhnimhnafkhbdqqknrxl.supabase.co

VITE_SUPABASE_ANON_KEY
[Get from Supabase → Settings → API → "anon public"]
```

### **5️⃣ Deploy**
- Click "Deploy"
- Wait 2-3 minutes
- Done! 🎉
